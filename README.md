## Julias Shop of Silver

A project in Sanity and Next.js, with Stripe as payment method
https://the-silver-shop-2a3g.vercel.app/

### Sanity (backend)
```bash
cd studio
npm i
npm run dev
```

### Next.js (frontend)
```bash
cd web
npm i
npm run dev
```
